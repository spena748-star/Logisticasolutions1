❓ Centro de ayuda

Aprende a usar Logistics PRO para optimizar tus cargas

1
Activar licencia

Ingresa la clave de licencia que recibiste. Formato: RENT-30-xxxxxxxx (30, 180 o 365 días). Una vez activada, la aplicación se desbloqueará.

2
Agregar pallets

Define dimensiones (pulgadas) y peso (libras). Marca "Apilable" si se pueden apilar verticalmente y define el máximo por columna.

3
Seleccionar remolque

Elige el tipo de trailer (Dry Van, Flatbed, Step Deck o Container). Puedes añadir capacidad extra si tienes permiso de sobrepeso.

4
Optimizar (Tetris)

Presiona "OPTIMIZAR CARGA". El sistema calculará automáticamente el acomodo tipo Tetris, apilamientos, centro de gravedad y carga por ejes según FMCSA.

5
Exportar resultados

Descarga un PDF con el informe técnico completo o guarda el historial de tus optimizaciones.

💡 Ejemplo práctico

Usa el botón "Ejemplo mixto" para cargar 3 pallets de diferentes tamaños y ver cómo el sistema calcula apilamientos, distribución en el trailer y advertencias de altura/peso.

🔐 Para administradores

En la pestaña "Aplicación" encontrarás un botón "Acceso administrador". La contraseña es admin2025. Allí podrás generar claves de licencia para tus clientes.
